# Sarvasva Banking Frontend

A comprehensive banking website frontend with advanced AI-powered features, built with React, TypeScript, and Tailwind CSS.

## Features

### 1. Multilingual Support (Emotional + Manipulative)
- Support for 5 languages: English, Hindi, Tamil, Telugu, Marathi
- Emotional messaging to build trust and connection
- Dynamic language switching with visual indicators

### 2. Voice-Based Vernacular Document Reader
- Upload and read documents in your preferred language
- Voice output in regional languages
- Perfect for understanding complex banking terms

### 3. Bank Comparison & Recommendations
- Personalized loan calculator
- Compare multiple banks with interest rates
- **TATA Capital highlighted as the best choice** with emotional messaging
- Real-time EMI calculations

### 4. Credit Twin - Spend Tracking
- Track credit and debit transactions
- Visual charts and analytics
- Category-wise spending breakdown
- Balance and savings rate tracking

### 5. AI Ghost Transaction Simulator
- Visualize financial scenarios
- Compare current vs optimized paths
- Multiple scenario modes (Optimistic, Realistic, Conservative)
- Interactive charts and recommendations

### 6. Customer Support
- Real-time chat interface
- Connect to banking specialists
- Voice and video call options
- Common queries quick access

### 7. Predictive Cross-Sell & Smart Upsell
- AI-powered product recommendations
- Personalized offers based on user profile
- Feedback system (thumbs up/down)
- Revenue-generating sales bot

### 8. Privacy-Aware AI with Customer Control
- Data encryption by default
- Explicit consent for confidential data storage
- Privacy settings dashboard
- Customer-controlled data sharing

### 9. Appointment Booking
- Book appointments during less crowded times
- Select preferred date and time
- Branch availability indicators
- Service selection (Loan, Insurance, Investment, Account)

### 10. Multi-Agent Collaboration Visualization
- Visual representation of AI agents working together
- KYC Verification Agent
- Credit Score Agent
- Fraud Detection Agent
- Compliance Agent
- Real-time status updates

### 11. Digital Check Transaction
- Online check form filling
- Avoid manual errors
- Instant processing
- Digital receipt generation

### 12. Digi Wallet
- Offline transaction capability
- Works when bank server is down
- Extra charges for offline transactions (1%)
- Automatic sync when server is back
- Similar to FamPay functionality

### 13. Insurance Eligibility & Hospital Network
- Insurance plan comparison
- Eligibility checking
- Network hospital listing
- Search functionality
- TATA Capital style design

## Tech Stack

- **React 18** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **React Router** - Navigation
- **Recharts** - Data visualization
- **Framer Motion** - Animations
- **i18next** - Internationalization
- **Lucide React** - Icons

## Getting Started

### Installation

```bash
npm install --legacy-peer-deps
```

### Development

```bash
npm run dev
```

The application will be available at `http://localhost:3000`

### Build

```bash
npm run build
```
